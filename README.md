# Spring MVC패턴 연습 위한 FX게시판

## console로 MVC패턴의 게시판을 구현해보았지만 약간의 부족함을 느껴 FX로도 한번 더 만들어봄.

## 구성: 
- application(스프링의 Start역할) , FXML(스프링의 프론트역할) , domain(Entity와 dto역할) , DAO(Repository와 Service역할) , Controller(Service와 Controller)
